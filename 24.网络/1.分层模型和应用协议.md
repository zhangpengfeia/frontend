# 分层模型和应用协议
## 分层模型
应用层：负责处理用户请求和响应，如HTTP、FTP等。
传输层：TCP,UDP等。
网络层：IP, 路由器。
数据链路层：MAC，交换机。
物理层：负责数据的物理传输，如电缆、光纤等。

## 应用层协议
URL: 固定格式字符串，表达了你从哪台计算机的哪个程序，寻找哪个服务，并注明获取服务的具体细节，以及用什么协议

服务器地址：http://localhost:5000/index.html 这个简称静态资源服务器
1. vscode 启动 list server 服务器
2. 访问 http://localhost:5000/index.html
