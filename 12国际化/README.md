# 国际化
国际化：是指在软件中考虑到了不同语言，不同文化，不同区域的需求
国际化程序：一般看移植性和通用性，意味着成本开销问题
```js
1.vue-i18n库，i18next，等
2.css处理
技术方案：
1.判断用户区域
    ip地址
    程序默认配置
    之前访问产生的cookie或localStorage缓存
2.用户切换地区/语言/货币/功能
3.国际化处理
    文案内容改变
    样式和外观
    图片内容改变
    本地化内容投放
4.css中国际化
5.服务器部署

浏览器默认语言
路由语言：
  path: /:locale? 区分
  beforeEnter: 设置路由导航首位，调用路由中间件函数，根据路由参数设置语言环境
```
